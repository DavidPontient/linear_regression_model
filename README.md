# linear_regression_model
My mission is to aim to predict crop yield based on environmental and farming conditions such as rainfall, temperature, and soil type.
Farmers often lack data-driven tools to estimate productivity before harvest.
This model will help provide quick predictions to support better agricultural planning.
The solution combines machine learning, an API, and a mobile app for accessibility.

# Mission
Help farmers understand how environmental factors affect crop production so they can increase their harvest.
